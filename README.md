# stuff-infra
Infra stuff for Stuff!

## Prerequisites

```shell 
chmod +x scripts/*.sh
```

### Create artifacts stack
```shell

```

### Update artifacts stack
```shell
./scripts/update-artifacts-stack.sh
```

### Delete artifacts stack
```shell
./scripts/update-artifacts-stack.sh
```
